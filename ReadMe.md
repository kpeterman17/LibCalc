LibCalc
=======
LibCalc is a C/C++ library used for creating calculators.

Functions
=========
add : Addition
sub : Subtraction
mult : Multiplication
div : Division

How to Use
==========
Refer to 'Demo.c', that contains my way of using it. It's nothing hard to understand, so I'm sure you'll figure it out.
